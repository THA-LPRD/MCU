Vielen Dank, dass Sie sich für einen LPRD-Display entschieden haben.

Sie haben 2 Möglichkeiten personalisierte Inhalte anzuzeigen:
Upload eines PNGs oder Design via HTML.



Hochladen von PNGs:

Bitte laden Sie ein PNG in dem exakten Pixelformat Ihres Displays hoch ( x px).
Achtung: wird ein PNG im falschen Pixelformat hochgeladen, so kann es nicht auf dem Display dargestellt werden!


    Eigenes Design mit HTML erstellen:

  1. HTML-Code in das Textfeld eingeben
    Achtung hierbei auf die Pixel-Auflösung des Displays ( x px): die Layoutgröße muss genau der Pixelanzahl entsprechen, sonst kann es nicht auf dem Display angezeigt werden!
  2. Eine Vorschau Ihres HTML-Designs können Sie im 2. Feld sehen - sie wird automatisch aktualisiert
  3. Wenn Sie zufrieden mit Ihrem Design sind, können Sie über den Upload-Button Ihr fertiges HTML-Layout an das Display schicken
  4. Nun müssen Sie nur noch einen kurzen Moment warten, bis das Display seinen Inhalt aktualisiert hat
Bitte geben Sie in das Textfeld Ihren HTML Code ein.
